# SIPPS-Web-Sistem-Informasi-Pelanggaran-Presensi-Siswa-
Sistem Informasi Manajemen Pelanggaran dan Presensi Siswa (SIPPS) berbasis Web menggunakan React &amp; TypeScript. Dirancang untuk memudahkan sekolah dalam memantau kedisiplinan siswa melalui sistem poin akumulatif dan rekapitulasi kehadiran secara real-time.
